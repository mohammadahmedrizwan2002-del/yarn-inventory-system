# Yarn Inventory & Order Management System

This is a simple prototype project built to manage yarn inventory, track orders, and monitor stock levels.

**Goal:**  
To solve real-world problems in textile trading by creating a digital system for inventory tracking and order processing.

**What it does (planned features):**
- Add and update yarn stock
- Track incoming and outgoing orders
- View summary of total inventory
- Record basic transaction history

**Status:**  
This repository shows my early progress and my journey into software development and open source.

---

**Why I built this:**  
I come from a business background in the textile industry and want to create tools that solve real operational problems. This project is one step in building practical software solutions connected to real business needs.
